# Release 1.72.0

This release includes tons of new features and bug fixes.

**Dashboard Layout**

You are now able to rearange the dashboard layout cards to your likings.

![Edit Dashboard Layout](https://media.discordapp.net/attachments/792479984454991872/1153005962043609152/image.png?width=1440&height=1029)

**Status Value (10-code) text color**

You are now also able to add a custom text color to your status values (10-codes).
A preview section was also added so you can preview the text and background color of the status value.

**Fullscreen Modals**

You can now "fullscreen" a modal.

**Other Improvements**

- Streamline 911-call & incident edit ability.
- Improvements to active units tables.
- Improved search for active units tables.
